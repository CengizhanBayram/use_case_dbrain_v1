# VoiceAgent_Proje

Projeye hızlı başlangıç:

1. Sanal ortam oluşturun: `python -m venv venv` ve etkinleştirin.
2. Gereksinimleri yükleyin: `pip install -r requirements.txt`.
3. `.env` dosyasına API anahtarlarınızı ekleyin.
4. `python app.py` ile uygulamayı başlatın (örnek).

Dizin yapısı ve açıklamalar dosyalarda mevcuttur.
